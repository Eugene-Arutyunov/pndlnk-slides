База ДКЦП

Ещё одна проверка

Мы научились проверять сходимость — корректную состыковку блоков модели друг с другом.

Но есть ещё одна проверка, и она же — ценность: нарратив.

Аргументы не просто связывают блоки — из них складывается связный рассказ о ценностном предложении.

Если аргументы не собираются в нарратив — скорее всего, модель ещё не до конца сошлась.

База ДКЦП

Шаблоны аргументов

Аргументы — это сообщения о взаимо­связи блоков ценностного предложения.

Каждый аргумент связывает не менее двух блоков ДКЦП.

Для разных пар и троек блоков есть языковые конструкции, которые помогают собрать аргумент.

{% include 'pndlnk.html' %}

База ДКЦП

Чему мы научились

  • Одна модель для очень разных команд — ДКЦП работает для любого ценностного предложения, независимо от специфики
  • Одна модель для себя и клиента — и для внутреннего понимания, и для коммуникации с заказчиком
  • Для любого ЦП — структура ДКЦП универсальна, потому что основана на инвариантах обмена
  • Мы научились специальному языку — языку блоков, сходимостей, аргументов и нарратива
  • И теперь не можем раз­видеть — когда начинаешь думать блоками ДКЦП, видишь структуру ценностного предложения повсюду
