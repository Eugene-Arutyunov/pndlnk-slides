База ДКЦП

Ещё одна проверка

Мы научились проверять сходимость по формальным критериям.

Есть ещё одна проверка на сходимость: нарратив. Менее формальная, но более безжалостная.

Нарратив — не только проверка, но ещё самостоятельная ценность и цель.

Покажем на себе.

База ДКЦП

Шаблоны аргументов

Аргументы — это сообщения о взаимо­связи блоков ценностного предложения.

Каждый аргумент связывает не менее двух блоков ДКЦП.

Для разных пар и троек блоков есть языковые конструкции, которые помогают собрать аргумент.

База ДКЦП

Чему научились

  • Одна модель для очень разных компаний — ДКЦП работает для любого ценностного предложения, независимо от специфики бизнеса.
  • Одна модель для себя и клиента — клиента понять проще, когда примеряешь на себя.
  • Успех продукта – не лотерея. Факторы, которые определяют, будет ли продукт продаваться, можно описать с помощью ДКЦП и обсуждать на его основе.
  • И теперь не можем раз­видеть — когда начинаешь думать блоками ДКЦП, видишь структуру ценностного предложения повсюду
